# 法學論文參考文獻格式審查

## 基本原則

1. **逐一列出**所有參考文獻，以利讀者查詢。
2. **分類臚列**：須區別「專書」、「期刊論文」、「專書論文」及「其他」等項目分別列出。
3. **排列順序**：文獻前加編號，**中文在前，其他外文在後**。
   - 中、日文：依作者姓氏**筆畫**排列先後
   - 外文（英、德等）：依作者姓氏**字母**順序排列
4. **頁碼必填**：期刊論文或專書中篇章，須列出起迄頁碼。

---

## 中、日文文獻格式

### 1. 專書

**格式：** 作者，書名，出版者，出版年月（西元）。

**版次規則：**
- 初版：**不需**標示版次，僅列出版年月
- 二版以上：須同時標示出版年月**及**版次

**範例：**
> 林錫堯，行政罰法，元照，2005年10月。
> 陳敏，行政法總論，自版，2009年9月6版。

### 2. 期刊論文

**格式：** 作者，論文名，期刊名，卷期別，起迄頁碼，出版年月（西元）。

**範例：**
> 李震山，德國抗制恐怖主義法制與基本權利保障，月旦法學雜誌，第131期，頁5-21，2006年4月。

### 3. 專書論文

**格式：** 作者，論文名，所載書名，出版者，起迄頁碼，出版年月（西元）。

**範例：**
> 蘇永欽，台灣的社會變遷與法律學的發展，載：當代法學名家論文集，法學叢刊，頁557-600，1996年1月。

## 英文文獻格式

### 1. 專書

**格式：** 作者姓，作者名，書名，出版社（版次，出版年）。

**說明：** 作者姓／名、書名，首字大寫。

**範例：**
> Hiller, Tim, Source Book on Public International Law, Routledge-Cavendish (2nd ed. 2000).

### 2. 期刊論文

**格式：** 作者姓，作者名，論文名，卷／期刊名／起頁（出版年）。

**說明：** 作者姓／名、論文名、期刊名，首字大寫。

**範例：**
> Reich, Charles A., The New Property, 73 Yale L.J. 733 (1964).

### 3. 專書論文

**格式：** 作者姓，作者名，論文名，專書名／起始頁碼，出版社，（編輯者，出版年）。

**說明：** 作者姓／名、論文名、專書名，首字大寫。

**範例：**
> Carre, Francoise J., Temporary Employment in the Eighties, in New Policies for the Part-Time and Contingent Workforce, 46, M.E. Sharpe (Virginia L. duRivage ed., 1992).

---

## 德文文獻格式

### 1. 專書

**格式：** 作者姓，作者名：書名，版次（Aufl.），出版年。

**範例：**
> Würtenberger, Thomas: Verwaltungsprozessrecht, 2. Aufl., 2006.

### 2. 期刊論文

**格式：** 作者姓，作者名：論文名，期刊名，出版年，起迄頁碼（S.）。

**範例：**
> Bettermann, Karl August: Über die Beiladung im Verwaltungsstreitverfahren oder vom Nutzen der Prozessvergleichung und einer allgemeinen Prozessrechtslehre, DVBl. 1951, S. 74-85.

### 3. 專書論文

**格式：** 作者姓，作者名：論文名，in: 編者姓（Hrsg.），書名，版次（Aufl.），出版年，起迄頁碼（S.）。

**範例：**
> Papier, Hans Jürgen: Recht der öffentlichen Sachen, in: Erichsen/Ehlers (Hrsg.), Allgemeines Verwaltungsrecht, 14. Aufl., 2010, S. 855-930.

## 操作流程：一步一步審查

### 先決條件

請使用者提供參考文獻清單（全部或部分皆可）後，再開始審查。

### 第一步：整體架構檢查

確認以下項目，提出問題並等待使用者回覆後，再進入第二步：
- 中文文獻是否排在外文文獻之前
- 各語言文獻是否依「專書→期刊論文→專書論文→其他」分類
- 各筆文獻前是否有編號
- 中、日文是否依作者姓氏筆畫排序；外文是否依作者姓氏字母排序

### 第二步：逐筆格式審查

依文獻類型逐一核對必要欄位，指出缺漏或格式錯誤，並提供修正範例。

### 第三步：一致性審查

確認同類文獻格式是否統一、年代是否全部使用西元紀年。

## 常見問題與修正方向

| 問題 | 修正方向 |
|------|----------|
| 中外文混排，未分類 | 中文在前，外文在後，分開排列 |
| 未依類型分項 | 依「專書→期刊論文→專書論文→其他」分類臚列 |
| 文獻前未加編號 | 逐筆補上編號 |
| 中、日文未依姓氏筆畫排序 | 依筆畫由少至多重新排列 |
| 外文未依姓氏字母排序 | 依 A 至 Z 重新排列 |
| 期刊論文或專書論文缺少頁碼 | 補充起迄頁碼 |
| 初版誤標版次 | 初版不標版次，刪除 |
| 二版以上未標版次 | 補充版次（如「2版」、「6版」） |
| 年代使用民國紀年 | 統一改為西元紀年 |
| 英文作者格式錯誤（名在前、姓在後） | 改為「姓，名」格式 |
| 英文書名、論文名、期刊名首字未大寫 | 補正各詞首字大寫 |
| 英文期刊論文誤套專書格式 | 須附卷數、期刊縮寫名、起頁及出版年 |
| 英文專書論文缺 in 或起始頁碼 | 補「in 專書名」及起始頁碼 |
| 中文專書論文缺「載：」或起迄頁碼 | 補「載：所載書名」及起迄頁碼 |
| 德文期刊論文頁碼未加 S. | 起迄頁碼前補「S.」 |
| 德文專書論文缺 in: 或 Hrsg. | 補「in: 編者姓（Hrsg.）」 |
| 德文文獻作者姓名間冒號缺漏 | 作者姓名後應接冒號 |
